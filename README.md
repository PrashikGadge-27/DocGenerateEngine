# DocGenerateEngine
An web app that takes data is excel format allow user to create an document template drag and drop there data varying column and bulk generate document for different data having same doc format.

HOW TO RUN THE CODE....
Download the zip, install packges.
warning!!!! --- if the editor is not working go to Editor.cshtml in that search for "  <script src="https://cdn.tiny.cloud/1/bwhs4ef0kqg1rtkb743fhuwscxn6e3y6h106goun4a0fkfft/tinymce/6/tinymce.min.js"
         referrerpolicy="origin"></script> "
          so go to tiny.cloud and generate one api key and paste it in here.
then run the code.


HOW TO USE..........
1. Go to upload screen upload an excel with data like [Book1.xlsx](https://github.com/user-attachments/files/24619889/Book1.xlsx). **having clear coloumn names in first row**
2. Go to Editor Screen now
                           1. first select from which file you want your parameters.
                           2. second make an document <img width="1784" height="899" alt="image" src="https://github.com/user-attachments/assets/3115b490-5f53-4eb7-89d6-95cd8b19a9d9" /> like this drag
                              and drop your parameters is desired positionin document.showen in image.
3. Then hit Generate document button give the any coloumn name in the dialogue box and procede.


OUTPUT..............
Now and zip file with documnents will get download, like [GeneratedDocs (1).zip](https://github.com/user-attachments/files/24620031/GeneratedDocs.1.zip) this having document generated.
